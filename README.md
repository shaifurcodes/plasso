# Plasso

> Any personality. One robot.

Plasso is the first AI persona robot platform. Download Lincoln, Einstein, or your brand mascot into one robot body. Fully offline. Swappable. Yours.

## What We're Building

- **Robot platform** — Jetson Orin Nano, 4-DOF servo system, standardized swappable head mount
- **Local AI** — Gemma running natively via llama.cpp. No cloud. No API costs. Sub-2 second response.
- **Persona system** — RAG-grounded historical and celebrity personas with voice, gestures, and character
- **Marketplace** — downloadable AI personas, creator SDK, 70% revenue share for creators
- **B2B licensing** — museums, education, celebrity management, brand mascots

## Stack

| Layer | Technology |
|-------|-----------|
| Hardware | Jetson Orin Nano 8GB + Feetech servo bus |
| AI inference | llama.cpp + Gemma 4 (fully local) |
| Speech to text | Whisper.cpp |
| Text to speech | Piper (local, voice-cloneable) |
| RAG | Qdrant (local vector store) |
| Framework | ROS2 Humble |
| Vision | OpenCV + IMX219 |
