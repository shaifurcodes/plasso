# Plasso Persona Format v0.1

A Plasso persona bundle contains:

persona-name/
├── manifest.json      # name, version, author
├── personality.json   # traits, tone, style
├── gestures.yaml      # servo gesture mappings
├── voice/             # Piper TTS voice model
└── corpus/            # RAG vector store snapshot
